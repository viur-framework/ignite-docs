Helper

ViUR Ignite has many little helper tools. See those on this page.

Floats & Clearfix

Add the class .f-left, .f-right or .f-none to an element to float it.

Add the class .clearfix to an element to clear the following content or create a .clear element which clear also.

Pointer

Add the class .pointer to an element to change the cursor to pointer on hover.

Shaddow

The class .shd or .txt-shd add a shaddow to the element.

I have a pretty box-shaddow!

I have a pretty text-shaddow!

Responsive Elements

Add the class .responsive-16by9 or .responsive-4by3 to an element to get the aspect ratio 16:9 or 4:3. It's very helpful if you embed Videos.
